Kyutai — Open Science AI Lab (kyutai.org), 26 public repos.

Key repos:
- moshi: Speech-text foundation model + full-duplex spoken dialogue framework. Uses Mimi neural audio codec.
- hibiki: Streaming speech translation (simultaneous translation), chunk-by-chunk real-time.
- jax-flash-attn3 / jax-flash-attn2: JAX bindings for flash-attention kernels.
- sphn: Python bindings for symphonia/opus — read audio formats, write opus.
- moshi-webrtc: PoC for running moshi/hibiki over WebRTC.
- yomikomi: Small Rust-based data loader.
