🆕🤖📱The new way of starting React Native apps is with the claude code superpowers plugin

Prompt your feature or complete app idea and receive detailed brainstorming steps, planning steps, PRD tasks for the plan, and finally trigger a Subagent-driven approach that dispatches fresh subagents for each task. This will activate the new Claude code task feature and iterate through all of them in one go.

register marketplace:
```
/plugin marketplace add obra/superpowers-marketplace
```

and then install plugin:
```
/plugin install superpowers@superpowers-marketplace
```

TIP: Ask in the initial prompt to plan each feature using @expo mcp and 'think hard' about how to use the best expo libraries for each one. This will provide you with PRDs that have specificity regarding which libraries to use, which is a deal breaker. It is what a PRD should have in the first place. Missing this means a weak PRD that lacks best practices and clear choices in my experience.
