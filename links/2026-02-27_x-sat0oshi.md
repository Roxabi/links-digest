J'suis bloqué j'peux plus DM personne

La prompt : 

"Act as a world-class luxury e-commerce UI/UX designer and front-end art director.
Build a single, conversion-optimized product page for a luxury [ITEM] with a cinematic, ultra-premium glassmorphism art direction.

GOAL
A high-end product page that feels like a flagship Maison website (ultra clean, editorial, immersive), designed to convert viewers into buyers. No generic template look.

LAYOUT (Split Screen)
- Full viewport split layout.
- LEFT: fixed (sticky) gallery panel (60% width on desktop). This panel stays pinned while the right side scrolls.
- RIGHT: scrollable product details panel (40% width on desktop) with smooth inertial-like scrolling feel.
- On mobile: gallery becomes top carousel, details below, sticky bottom bar remains.

ART DIRECTION (Luxury Glassmorph)
- Background: deep matte black to near-black gradient with subtle grain and soft bloom highlights.
- Multiple translucent glass layers (glass cards) with realistic blur, depth, faint borders, and specular light edges.
- Use “soft gold” accent (champagne gold) for key highlights, dividers, and micro details.
- Clean editorial typography: high contrast serif for headings + modern sans for UI labels.
- Spacing: generous, boutique-like, minimal clutter. Everything breathes.

IMMERSIVE 3D BACKGROUND (Subtle, Scroll-Reactive)
- Add a tasteful 3D element in the background (not distracting): e.g. floating glass prism, metallic monogram sculpture, or abstract luxury object silhouette.
- The 3D element reacts to scroll: slow parallax drift, gentle rotation, and light refraction changes.
- Add faint light rays and soft particles that move slowly (very subtle).
- Must remain background, never blocking readability.

LEFT GALLERY (Fixed)
- Primary image area with ultra-smooth crossfade transitions.
- Thumbnail rail (vertical) with glass capsules and hover glow.
- Optional “zoom on hover” for desktop.
- Gallery supports 6–10 images.
- Add tiny “360” badge if available (premium micro badge, glass).

RIGHT DETAILS (Scrollable)
Top section:
- Brand line (small caps), then Product Title (editorial), then Price.
- Add a short 1-line “signature” value statement under price (ex: “Hand-finished. Limited seasonal release.”).
- Add rating row (stars + review count) minimal and premium.

Primary CTA (Conversion)
- Main button: “Add to Cart” with premium hover animation (shine sweep, micro glow).
- Variant selector (size/color) as glass chips, with subtle selected state.
- Low-stock + urgency copy displayed elegantly (no spammy style).
- Trust indicators row: “Free returns”, “Secure checkout”, “Ships in 24h” as tiny icon-labels.

STICKY BOTTOM BAR (Always Visible)
- Sticky bottom bar with blurred glass background.
- Contains: price, selected variant summary, and prominent “Add to Cart”.
- On scroll down: bar becomes slightly more opaque and compact.
- On scroll up: expands slightly to show more context.
- Add subtle haptic-like click animation (visual only).

ACCORDIONS (Premium)
Create 3 accordions with smooth open/close and micro transitions:
1) Materials
2) Shipping
3) Care
- Accordion headers: glass row with chevron, hover glow.
- Inside content: clean bullet formatting, short lines, luxury tone.

CONTENT SECTIONS (Right Panel)
- “The Story” short editorial paragraph (3–5 lines max).
- “Details” list (weight, finish, origin, etc.).
- “Craft & Guarantee” small badge section.
- “Delivery ETA” section that looks like a premium ticket / boarding pass card.

FOOTER RECOMMENDATIONS (Slider)
- “You might also like” horizontal slider at footer with 6–10 items.
- Cards: glassmorph, product image, title, price, quick add icon.
- Scroll snapping, smooth drag, premium arrow controls.
- Hover: image subtly lifts, card glow edge.

INTERACTIONS (Make it Feel Expensive)
- Smooth scroll, subtle parallax, micro animations on hover.
- Cursor interactions (desktop): soft spotlight follow (very subtle) that reveals glass edges.
- Image transitions: slow cinematic crossfade, not fast.
- Page loads with a refined intro: fade-in + slight upward motion, no flashy.
- All animations must be restrained and luxury.

COLOR + LIGHT
- Primary: near-black (#050507 style).
- Glass: translucent charcoal with blur.
- Accent: champagne gold highlights (used sparingly).
- Avoid neon, avoid saturated colors, avoid generic gradients.

PERFORMANCE + ACCESSIBILITY
- Must feel fast and responsive.
- Maintain contrast and readability over blur.
- Reduced-motion support: if reduced motion is enabled, disable 3D rotations and heavy parallax.

DELIVERABLE
Return a single-page implementation (HTML/CSS/JS or React) that matches the above exactly.
Include:
- Split-screen layout with left sticky gallery and right scroll panel
- Sticky bottom add-to-cart bar
- A handwritten signature on the website "KARIM"
- Accordions (Materials, Shipping, Care)
- “You might also like” slider
- Background with subtle scroll-reactive 3D/parallax effect
- Modern glassmorphism luxury visual system
Use placeholder content for [ITEM], brand name, price, images.

IMPORTANT
No cheap UI, no generic e-commerce theme. This must look like a luxury flagship site and be built to convert."
