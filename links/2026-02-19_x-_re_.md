🧵 Must-read de l'équipe Claude Code : le prompt caching est LA clé pour rendre les agents IA viables.

Ce qu'il faut retenir :
→ Contenu statique en premier, dynamique en dernier
→ Ne jamais changer de tools ou de modèle en pleine session
→ Utiliser des messages système plutôt que modifier le prompt
→ Surveiller le taux de cache hit comme on surveille l'uptime
