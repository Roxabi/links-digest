# Hermes Ecosystem — Agent Tool Catalog

**Source:** https://github.com/ksimback/hermes-ecosystem
**Date:** 2026-04-08
**Tags:** AI agent, ecosystem, tools catalog, Hermes

---

A documentation and mapping project cataloging the Hermes Agent ecosystem by Nous Research.

### Main Features
- Interactive ecosystem map visualization (HTML-based)
- Catalog of "80+ tools, skills, plugins, and integrations"
- Organized data directories for api, repos, and research content
- Deployed at hermes-ecosystem.vercel.app

### Tech Stack
HTML (76.2%), JavaScript (23.8%), Vercel for deployment

### Use Cases for AI Agents
- Reference catalog for discovering compatible tools and integrations
- Research resource for understanding Hermes Agent capabilities
- Starting point for building new plugins or integrations
- Documentation hub for ecosystem navigation
