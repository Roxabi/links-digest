# pi-fff — FFF-Powered Fuzzy Find & Grep Extension for pi

**Source:** https://github.com/SamuelLHuber/pi-fff
**Date:** 2026-04-10

---

-|
| `grep` | Content search with pagination cursor support |
| `find` | Fuzzy file name search, frecency-ranked |
| `multi_grep` | OR-logic multi-pattern search |

## Tech Stack

TypeScript 100%, uses FFF Rust library, Bun package manager

## Data Storage

`~/.pi/agent/fff/frecency.mdb` and `history.mdb`

## Installation

```bash
pi install git:github.com/SamuelLHuber/pi-fff        # global
pi install -l git:github.com/SamuelLHuber/pi-fff     # project-local
pi install git:github.com/SamuelLHuber/pi-fff@v0.2.3 # pinned
```

## Commands

- `/fff-health` — Check status
- `/fff-rescan` — Force re-index
- `/fff-mode both|tools-only` — Switch modes
