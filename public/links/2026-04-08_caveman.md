# Caveman — Token Compression Skill for AI Agents

**Source:** https://github.com/JuliusBrussee/caveman
**Date:** 2026-04-08

---

A Claude Code skill/plugin that makes AI agents respond in "caveman-speak" — terse, fragmented language that cuts approximately 65-75% of output tokens while preserving full technical accuracy.

### Main Features
- **Intensity levels:** Lite, Full, Ultra (from professional brevity to maximum compression)
- **Wenyan mode:** Classical Chinese literary compression variant
- **caveman-commit:** Generates terse conventional commit messages (≤50 char subjects)
- **caveman-review:** One-line PR comments without filler text
- **caveman-compress:** Compresses memory files like CLAUDE.md to reduce input tokens by ~45%

### Tech Stack
Python (86%), JavaScript (7.2%), Shell (6.8%)

### Use Cases
- Reducing LLM token costs and API expenses
- Faster response generation (fewer tokens = faster output)
- More readable, concise technical responses
- Efficient code reviews and commit messages
- Compressing context files loaded at session start

### Installation
Works with multiple AI agents including Claude Code, Cursor, Copilot, Windsurf, Cline, and Codex via `npx skills add JuliusBrussee/caveman` or plugin marketplace.
